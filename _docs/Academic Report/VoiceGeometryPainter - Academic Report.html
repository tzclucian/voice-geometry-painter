Abstract

Voice Geometry Painter is an application that allows people (especially Mathematics teachers and pupils, but not only) to draw geometric shapes by speaking certain commands.

The purpose of the application is to allow people that have some motion disabilities to draw and manipulate geometric shapes, helping them to study geometry. We think this could be a great help for mathematics teachers that have a motion disability and cannot draw/write at the whiteboard.

The interaction with the application is simple, based on short voice commands like draw <shape_name> <simple_parameters> for drawing, and even simpler commands for other operations, like clear board.

Built upon the following technologies

  • JSXGraph - Cross-browser javascript library for interactive geometry, plotting and visualization.
  • Bootstrap - HTML, CSS, and JS framework for responsive web UI development.
  • Knockout - Javascript library for dynamic UI
  • FileSaver.js - implements the HTML5 W3C saveAs() FileSaver interface
  • Blob.js
  • Canvas-ToBlob.js
  • Dropbox datastore js sdk
  • HTML5 Speech Recogntion API

Features developed

  • Drawing geometric shapes
    • Points
    • Segments
    • Triangles (Isosceles, Rectangular, Equilateral and irregular)
    • Quadrilaterals (Parallelogram, Rectangle, Square and irregular)
    • Circles
  • Select/Deselect/Delete shapes
  • Show properties of a shape
  • Set Fill/Line colors
  • Set Opacity
  • Set line size
  • Download on local drive
  • Share on Dropbox
  • Set board title
  • Clear board
  • Undo/Redo

Drawing Area

The drawing functionality is implemented in DrawingContext.js file and uses jxgraph library.


Speech Recognizer

The speech recognition component uses an instance of WebkitSpeechRecognition engine, subscribing a set of callbacks for recognition events like OnResult, OnEnd, etc.

In addition, having a set of commands as input, the component creates a grammar, which is later used as a state machine in the recognition step. This functionality is implemented in CommandParserStream.js.


Command Parsing

Each command is defined as an object which has a parametrized regular expression property and an execute method. When the Command Parser processes a new phrase, it tries to match each command from it's list, and in case of success, the parser calls it's execute method, which contains the logic of the command execution.







User Interaction

Speech commands

The user can draw any shape using voice commands and also can issue other actions like Undo, Download, Share on Dropbox and so on.


Mouse Interaction

For users that cannot speak the language, or cannot speak at all, the application offers a mouse compatible interface. This interface consists of a tool-bar with all the necessary commands.
