Q2: How do I learn to use the app? Question 2    
O1: I am asked if I wish a tutorial and I accept.   Option 1  
C1: The app explains me what commands I can use for different tasks: select different tools, select drawn objects, move brush and so on.     Criteria 1
C2: The app asks me to give some commands in order to confirm that I understood.     Criteria 2
C3: Towards the end I am presented a simple drawing and asked to give the specific commands in order to draw it. I get feedback after each command.     Criteria 3
O2: I don't accept the tutorial, but I can anytime go to "Help" section.   Option 2  
C1: I can navigate through various help sections.     Criteria 1
C2: The documentation is structured as a static html.     Criteria 2
Q3: How do I draw using voice? Question 3    
O1: Animated brush movement controled by voice commands.   Option 1  
C1: I ask to place the pointer to a certain position. I am presented a grid with numbers in each cell and I speak the number of the cell where I want to point my cursor in the top-left corner.      Criteria 1
C2: I specify which direction I want the pointer to go with commands like: "up", "down", "left", "right", "up-right", "up-right-curve" etc.. When I speak command "go", the pointer will move in the selected direction, with a certain speed (which I can change any time). The pointer will stop when I speak the command "stop".      Criteria 2
C3: In order to enable drawing on canvas I can speak command "let brush down", and "pick up brush" for disabling painting and alowing movement without painting.     Criteria 3
O2: Voice commands sequence for specific drawing actions.   Option 2  
C1: I can set the pointer to any position on the canvas by speaking the x,y coordinates.     Criteria 1
C2: I can speak something like "draw line", followed by the end point coordinates.     Criteria 2
C3: I can enable an orienting numbered grid to guide myself on coordinates.     Criteria 3
C4: I can change the end point until I find the perfect drawing for my line. I can give commands like "move left by 3 pixels" for the end point, and this will trigger a redraw to my line, with the end point in the new position).     Criteria 4
